O que é MeteorJS?

  • O Meteor é uma plataforma JavaScript full-stack para o desenvolvimento de aplicativos móveis e web modernos. O Meteor inclui um conjunto chave de tecnologias para a construção de aplicativos Client Side, uma ferramenta de construção e um conjunto de pacotes da comunidade Node.js e JavaScript.
  • O Meteor permite que você desenvolva em uma linguagem, JavaScript, em todos os ambientes: servidor de aplicativos, navegador da web e dispositivo móvel.
  • O Meteor usa dados na rede, o que significa que o servidor envia dados, não HTML, e o cliente os renderiza.
  • O Meteor abraça o ecossistema, trazendo as melhores partes da comunidade extremamente ativa de JavaScript para você de uma forma cuidadosa e ponderada.
  • O Meteor fornece reatividade Fullstack, permitindo que sua UI reflita perfeitamente o verdadeiro estado do mundo com esforço mínimo de desenvolvimento.
  • Entre outras vantagens do Meteor estão sua arquitetura, que dinamiza o código e acelera o processo de desenvolvimento, a facilidade em desenvolver interfaces modernas de design amigável, com pacote nativo de integração ao Bootstrap, responsividade para diversos tipos de resolução, seja desktop ou mobile, além das vantagens óbvias de uma plataforma de código aberto.

  • Vantagens

  • Mobile ready: Basta criar sua webapp e em poucos comandos, você consegue fazer um build do front-end da aplicação para Android e ou iOS.
  • Hospedagem free para aplicações pequenas: Você pode lançar uma versão beta ou staging de sua aplicação através da infraestrutura free do Meteor
  • Tem seu próprio package manager: Apesar do Meteor ser escrito em Node.js, ele possui seu próprio package manager, oficialmente conhecido pelo nome Atmosphere.
